PrintPro - Custom Online Printing Platform

This is a complete React-based frontend for a custom online printing platform similar to UPrinting. The application is fully functional with product catalog, customization, and cart management.

🛍️

Product Catalog

Browse products by category with filtering, search, and responsive grid layout.

⚙️

Customization

Dynamic product customization with real-time pricing updates and file upload.

🛒

Shopping Cart

Full cart management with quantity controls, pricing, and checkout flow.

🚀 Getting Started

Prerequisites:

  • • Node.js (LTS version)
  • • npm or yarn
  • • Modern web browser

Quick Start:

npm install
npm start

Note: This is a static demo. To see the full interactive application, please install Node.js and follow the setup instructions in SETUP_INSTRUCTIONS.md
